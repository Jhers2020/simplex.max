Algoritmo metodo simplex de maximizacion

El algoritmo simplex nos permite realizar una asignación optima de recursos asignando valores a las variables que optimizen
el rendimiento maximizando utilidades y miniimizando costos.


Esta aplicación resuelve problemas de asignacion de recursos atravez del metodo simplex para maximizacion con restricciones
del tipo menor que (<=). Este programa tiene la capacidad de resolver asignaciones con "n" variables y "r" restricciones.
